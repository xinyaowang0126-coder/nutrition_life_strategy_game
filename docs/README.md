# 文档目录

本目录收纳《撑过这一周》的设计、流程规范、素材说明和项目概述资产。

## 结构

- `design/`：完整 GDD、Godot 4.7 实现路径、早期玩法设计工作稿。
- `process/`：CodeX 方案审查与 Godot MCP 协作规范。
- `assets/`：MVP 素材清单与生成说明。
- `project_overview_assets/`：项目概述 `.docx` 使用的插画、UI 参考图和生成源图。
- `撑过这一周_项目概述与系统预览.docx`：面向展示和沟通的项目概述文档。

运行时卡片数据位于 `../life-strategy/data/cards/`，不是文档资产。

## 临时目录

文档渲染或格式检查可能会生成 `.tmp_render/`、`render_tmp_workaround/`、`tmp_render_plain/`、`rendered_project_overview/`。这些目录已被 `.gitignore` 忽略，可清理后再生成。
