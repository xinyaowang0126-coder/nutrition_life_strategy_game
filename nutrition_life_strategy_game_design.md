# 《撑过这一周》游戏设计说明（工作稿）

> 工作名暂定：**《撑过这一周》** / **《好好吃饭》** / **《今天也要撑住》**  
> 类型：生活策略 + 卡牌构筑 + 饮食配餐 + 轻度资源管理  
> 关键词：饮食、心理、预算、时间、精力、生活压力、角色差异、非说教式健康表达

---

## 1. 核心概念

本游戏不是让玩家扮演营养顾问，也不是单纯完成“健康饮食评分”的工具型游戏。

玩家扮演的就是**角色本人**：一个正在经历某段现实生活压力的人。

玩家需要在有限天数、有限预算、有限时间和有限精力下，通过吃饭、休息、学习/工作、购物、运动、社交和自我调节等行动，尽量让自己**撑过这一段生活**。

游戏底层有营养和心理健康逻辑，但表层目标不直接强调“健康达标”，而是更生活化，例如：

- 撑过考试周
- 熬到项目上线
- 用有限生活费过完两周
- 在调整饮食的同时不要被计划反噬
- 在压力和疲劳中维持基本生活节奏

核心体验不是“追求完美健康”，而是：

> **在不理想的现实条件下，做出相对可持续的选择。**

---

## 2. 设计愿景

### 2.1 玩家体验目标

希望玩家在游玩时产生以下感受：

- “我不是在做营养题，而是在过一段真实的生活。”
- “奶茶、外卖、方便面不一定只是错误，有时它们是短期撑住自己的方式。”
- “健康食物也不是万能答案，如果压力、预算和精力崩了，计划也无法持续。”
- “最好的结果不是完美，而是稳定、可持续、还能继续生活。”

### 2.2 游戏气质

游戏应避免说教、惩罚和道德化表达。

不应反复提示玩家：

- 你不应该吃这个
- 这个不健康
- 你失败了

更理想的表达是：

- 今天有点难，但你还是做了一些不错的选择
- 这顿不完美，但它帮你撑过了晚上
- 偶尔吃喜欢的东西没关系，关键是整体节奏
- 你吃得很“正确”，但压力已经太高了，这样可能很难持续

---

## 3. 单局结构

游戏以固定天数为一局。

建议主要模式包括：

| 模式 | 天数 | 说明 |
|---|---:|---|
| 短局 | 7 天 | 节奏快，适合反复尝试不同策略 |
| 标准局 | 14 天 | 更能体现习惯、状态波动和长期代价 |
| 挑战局 | 7 或 14 天 | 加入考试周、加班周、预算紧张等特殊压力 |

每局开始时，玩家选择或解锁一个角色。每个角色有不同的生活背景、初始状态、可用行动、饮食偏好和专属压力。

一局的表层目标可以是：

- 距离考试还有 7 天，尽量撑到最后
- 项目 14 天后上线，别让自己彻底崩盘
- 这两周只有固定生活费，尽量稳定过完
- 训练周期结束前，保持训练恢复和生活节奏

---

## 4. 每日流程

每天由若干个行动节点构成。具体可以抽象为：

1. 早晨
2. 上午
3. 中午
4. 下午
5. 晚上
6. 深夜

也可以简化为：

1. 早餐
2. 午餐
3. 晚餐
4. 行动选择
5. 晚间结算

每一天玩家需要在以下事情之间做选择：

- 吃什么
- 要不要省钱
- 要不要节省时间
- 要不要复习/工作/训练
- 要不要休息
- 要不要社交
- 要不要点外卖
- 要不要用某种方式安慰自己

一天结束后进行结算，更新角色状态，并生成第二天的影响。

例如：

- 睡得太晚，第二天精力下降
- 连续外卖，饮食负担上升
- 三餐稳定，饱腹感和心情更稳
- 预算过低，压力上升
- 复习进度不足，考试压力增加

---

## 5. 实时指标系统

### 5.1 设计原则

实时指标不能太复杂。玩家实时看到的应该是“生活状态”，而不是营养报表。

底层可以计算热量、蛋白质、脂肪、糖、钠、膳食纤维、微量营养等，但这些不应全部常驻显示。

主界面应以少量直接影响决策的指标为主。

---

### 5.2 主界面常驻指标

建议常驻显示以下指标：

| 指标 | 说明 |
|---|---|
| 稳定度 | 总体状态值，由多个底层因素复合得出，可理解为“还能不能撑住” |
| 余额 | 本周/本局剩余生活费 |
| 时间 | 当天剩余行动格 |
| 精力 | 当前能否执行高消耗行动，如做饭、运动、复习、加班 |
| 心情 | 心理状态，影响冲动选择、安慰食物收益和自控能力 |
| 饱腹感 | 当前饥饿程度，影响下一餐选择和冲动进食风险 |

示例 UI 文案：

```text
第 5 天 / 7 天    晚上

稳定度：62（有点危险）
余额：￥34
时间：■■□
精力：偏低
心情：低落
饱腹感：不足

今日状态：压力偏高｜饮食负担略高
```

---

### 5.3 稳定度

稳定度是一个表层总体指标，类似生命值，但不等同于传统血量，也不应显得过于医学化。

它代表角色整体是否还能维持生活节奏。

稳定度可以由以下底层因素复合得出：

| 底层因素 | 作用 |
|---|---|
| 饮食结构 | 影响身体状态和长期稳定性 |
| 心理状态 | 影响能否继续执行计划 |
| 精力/疲劳 | 影响行动能力 |
| 饱腹感 | 影响冲动进食和情绪稳定 |
| 睡眠 | 影响第二天精力、压力和心情 |
| 饮食负担 | 高油、高盐、高糖、暴饮暴食等累积压力 |
| 预算安全度 | 余额过低会增加压力和限制选择 |
| 角色目标进度 | 如复习进度、项目进度、训练恢复等 |

示例公式思路：

```text
稳定度 =
饮食状态 × 0.25
+ 心理状态 × 0.25
+ 精力状态 × 0.15
+ 睡眠状态 × 0.10
+ 饮食负担控制 × 0.10
+ 角色目标进度 × 0.10
+ 预算安全度 × 0.05
```

不同角色可以有不同权重。

例如学生更看重复习进度、压力和睡眠；健身角色更看重训练恢复、蛋白供给和能量是否足够；减脂角色更看重饮食稳定、心理状态和反弹风险。

---

### 5.4 半隐藏或每日结算指标

以下指标不一定常驻显示，可以在每日结算、状态详情或卡牌详情中出现：

| 指标 | 表层表达 |
|---|---|
| 饮食状态 | 最近吃得是否稳定、是否有结构问题 |
| 饮食负担 | 油盐糖和暴饮暴食累积压力 |
| 压力 | 当前生活压力 |
| 睡眠状态 | 影响第二天精力和心情 |
| 自控力 | 影响高诱惑食物出现率或选择成本 |
| 饮食焦虑 | 特定角色重要，过高会降低心理状态 |

不建议主界面实时展示过多营养细项，例如蛋白质、钠、糖、脂肪等。它们可以放在详情页、图鉴页或高级模式中。

---

## 6. 经济系统

### 6.1 是否需要经济系统

建议加入经济系统，因为现实中的饮食选择不是纯粹的营养最优问题，而同时受到钱、时间、精力、便利性、情绪和环境影响。

如果没有经济系统，玩家很容易永远选择看起来最健康的组合，策略深度会下降。

经济系统的目标不是做复杂经营，而是制造生活限制。

---

### 6.2 经济资源

经济系统建议使用生活化表达：

- 生活费
- 余额
- 餐费预算
- 本周预算

不建议使用传统 RPG 的“金币”表达，除非游戏整体采用明确的夸张风格。

示例：

```text
本周生活费：￥120
已度过：3 / 7 天
今日已花：￥18
当前余额：￥64
```

---

### 6.3 钱与其他资源的关系

钱不只是买食物，也是在购买便利、时间和心理安慰。

| 选择 | 花费 | 好处 | 代价 |
|---|---:|---|---|
| 自己做饭 | 低 | 饮食更可控 | 消耗时间和精力 |
| 食堂 | 低/中 | 稳定、方便 | 选择有限 |
| 外卖 | 中/高 | 省时间、满足感高 | 饮食负担上升 |
| 轻食 | 高 | 饮食状态较好、省时间 | 贵，满足感不一定高 |
| 便利店 | 中 | 快速 | 饮食结构不完整 |
| 奶茶甜品 | 中 | 心情恢复快 | 糖压力和预算压力上升 |
| 方便面 | 低 | 便宜、快、能救急 | 营养单一，饮食负担上升 |

---

### 6.4 库存与采购

可以加入轻量库存系统，但不宜过重。

建议形式：

- 采购一次获得未来几餐可用的食材卡
- 批量购买更便宜，但会占用当前预算
- 新鲜食材可能有保质期
- 耐放食物更稳定，但营养可能单一

示例：

| 采购选项 | 效果 |
|---|---|
| 买一盒鸡蛋 | 获得 3 张“鸡蛋”食物卡 |
| 买水果 | 获得 2 张“水果”食物卡，提升心情和微量营养 |
| 买方便面 | 获得 3 张“方便面”卡，便宜耐放，但饮食负担较高 |
| 买蔬菜 | 获得 2 张“蔬菜”卡，保质期较短 |

---

## 7. 行动系统

### 7.1 基本原则

不同角色不应拥有完全相同的行动池。

角色差异不仅体现在数值上，更应该体现在：

- 能做什么
- 不能做什么
- 做同一件事的代价不同
- 同一食物或行为对不同角色的效果不同

例如学生可以复习，但不一定能复杂做饭；上班族可以做饭，但下班后精力成本更高；健身角色可以训练，但训练后需要更多恢复和饮食支持。

---

### 7.2 行动消耗

每个行动至少消耗一种资源：

- 钱
- 时间
- 精力
- 心情
- 自控力

示例：

| 行动 | 钱 | 时间 | 精力 | 效果 |
|---|---:|---:|---:|---|
| 复习 | 0 | 1 | 1 | 复习进度上升，压力可能上升 |
| 简单做饭 | 低 | 1 | 1 | 饮食状态上升 |
| 点外卖 | 中 | 0 | 0 | 省时间，心情上升，饮食负担上升 |
| 买奶茶 | 中 | 0 | 0 | 心情上升，糖压力上升 |
| 运动 | 0 | 1 | 1 | 压力下降，但疲劳上升 |
| 小睡 | 0 | 1 | 0 | 精力恢复，可能影响进度 |
| 早睡 | 0 | 1 | 0 | 第二天精力恢复，放弃深夜行动 |

---

## 8. 角色系统

每个角色应包括：

- 背景处境
- 表层目标
- 初始状态
- 饮食偏好
- 可用行动池
- 不可用或受限行动
- 专属指标
- 专属事件
- 稳定度权重
- 多结局文本

---

### 8.1 角色一：熬夜学生 / 研究生

#### 表层目标

撑过考试周或论文截止周。

#### 初始状态

- 压力较高
- 睡眠不足
- 预算有限
- 容易点外卖
- 奶茶和咖啡诱惑较强
- 蔬菜摄入不足

#### 专属指标

- 复习进度 / 论文进度

#### 可用行动

| 行动 | 说明 |
|---|---|
| 复习 | 提高目标进度，但消耗时间和精力，可能增加压力 |
| 去食堂 | 低成本、较稳定，但选择有限 |
| 点外卖 | 省时间和精力，可能提升心情，但增加饮食负担 |
| 买奶茶 | 快速提升心情，但增加糖压力和花费 |
| 便利店买饭 | 快速，营养一般 |
| 简单冲泡 | 便宜省时，例如燕麦、泡面、速食粥 |
| 小睡 | 恢复精力，但占用时间 |
| 早睡 | 恢复第二天状态，但减少深夜复习时间 |
| 散步 | 降低压力，消耗时间 |

#### 受限行动

- 不能复杂做饭，除非特定场景解锁
- 备餐能力弱
- 采购能力弱或受限

#### 玩法重点

学生不是吃得最健康就赢。若饮食很好但复习进度不足，稳定度仍会下降。玩家必须在复习、睡眠、饮食和预算之间取舍。

---

### 8.2 角色二：加班上班族

#### 表层目标

撑到项目上线。

#### 初始状态

- 时间紧张
- 晚餐不规律
- 外卖依赖较高
- 疲劳累积
- 工作进度压力大

#### 专属指标

- 项目进度

#### 可用行动

| 行动 | 说明 |
|---|---|
| 加班 | 项目进度上升，但压力和疲劳上升 |
| 公司食堂 | 稳定、方便，选择有限 |
| 点外卖 | 省时间，但饮食负担增加 |
| 下班买菜 | 获得食材，消耗时间和精力 |
| 简单做饭 | 饮食状态较好，但疲劳高时成本增加 |
| 运动 | 降低压力，但消耗时间和精力 |
| 早睡 | 恢复精力，但可能牺牲项目进度 |
| 聚餐应酬 | 社交和心情上升，但饮食负担上升 |

#### 玩法重点

这个角色的核心矛盾是时间和精力。点外卖不一定是错误，有时是为了保护项目进度和休息时间。

---

### 8.3 角色三：减脂上班族

#### 表层目标

撑过 14 天的饮食调整期，不要被计划反噬。

#### 初始状态

- 饮食焦虑高
- 容易过度节食
- 晚上可能暴食
- 工作精力受饮食影响
- 对体重波动敏感

#### 专属指标

- 饮食焦虑 / 反弹风险

#### 可用行动

| 行动 | 说明 |
|---|---|
| 规律正餐 | 稳定饱腹感和心理状态 |
| 控制饮食 | 短期降低饮食负担，但过度会增加焦虑和反弹风险 |
| 点轻食 | 饮食状态较好，但价格高，满足感因人而异 |
| 称体重 | 获得反馈，但可能提高饮食焦虑 |
| 轻运动 | 心情上升，压力下降 |
| 吃喜欢的食物 | 心理恢复，饮食负担可能上升 |
| 朋友聚餐 | 心情上升，但计划被打乱 |

#### 玩法重点

不能设计成“越少吃越好”。如果连续能量过低，会导致心情下降、精力下降、反弹风险上升，甚至触发夜间暴食事件。

---

### 8.4 角色四：健身大学生

#### 表层目标

撑过训练周期。

#### 初始状态

- 运动量高
- 蛋白需求高
- 预算有限
- 饮食容易单调
- 容易忽视蔬菜水果

#### 专属指标

- 训练恢复

#### 可用行动

| 行动 | 说明 |
|---|---|
| 训练 | 提升训练进度，但增加疲劳和营养需求 |
| 食堂加餐 | 便宜补充能量 |
| 买蛋白食物 | 改善恢复，但花费较高 |
| 备餐 | 提升饮食可控性，消耗时间 |
| 休息拉伸 | 降低疲劳 |
| 社交聚餐 | 心情上升，但饮食计划可能被打乱 |

#### 玩法重点

不能只堆蛋白。训练恢复还需要能量、睡眠、蔬果、微量营养和预算平衡。

---

### 8.5 角色五：情绪性进食者

#### 表层目标

撑过情绪低谷的一周。

#### 初始状态

- 心情波动较大
- 压力高
- 甜食和零食诱惑强
- 饮食规律差
- 自控力不稳定

#### 专属指标

- 情绪波动 / 冲动进食风险

#### 可用行动

| 行动 | 说明 |
|---|---|
| 稳定正餐 | 改善饱腹感和情绪稳定 |
| 吃甜食 | 快速提升心情，但可能带来后续波动 |
| 散步 | 降低压力 |
| 联系朋友 | 提升心理状态，消耗时间 |
| 早睡 | 恢复第二天状态 |
| 心理调节 | 降低饮食焦虑或冲动风险 |
| 点外卖 | 快速获得满足感，但饮食负担上升 |

#### 玩法重点

不应鼓励完全压制甜食。少量、计划内的安慰食物可以成为合理策略。

---

## 9. 卡牌系统

游戏可以采用卡牌作为行动和食物的主要载体。

卡牌分为三类：

1. 食物卡
2. 行为卡
3. 心理卡

---

### 9.1 食物卡

食物卡代表一次饮食选择或食材。

每张食物卡可以包含：

- 花费
- 时间消耗
- 精力消耗
- 饱腹感影响
- 心情影响
- 饮食负担影响
- 角色偏好加成
- 隐藏营养属性

示例食物卡：

| 食物卡 | 可能效果 |
|---|---|
| 米饭 | 便宜、提供饱腹感和能量 |
| 燕麦 | 饱腹感高，饮食负担低 |
| 鸡蛋 | 高性价比蛋白 |
| 豆腐 | 便宜、稳定、适合省钱健康餐 |
| 青菜 | 改善饮食状态，花费低，但满足感因角色而异 |
| 水果 | 改善心情和饮食状态 |
| 奶茶 | 心情快速上升，糖压力和花费上升 |
| 炸鸡 | 高满足感，饮食负担上升 |
| 方便面 | 便宜、省时，但营养单一 |
| 轻食沙拉 | 饮食状态好，但贵，饱腹和满足感不一定稳定 |

---

### 9.2 行为卡

行为卡代表生活行为。

示例：

| 行为卡 | 效果 |
|---|---|
| 早睡 | 第二天精力恢复，压力下降 |
| 小睡 | 当前精力恢复，占用时间 |
| 复习 | 目标进度上升，消耗精力，可能增加压力 |
| 加班 | 项目进度上升，疲劳和压力上升 |
| 运动 | 压力下降，疲劳上升，需要饮食支持 |
| 采购 | 获得未来可用食材卡 |
| 备餐 | 未来几餐成本降低，当前消耗时间和精力 |
| 喝水 | 小幅降低饮食负担，提升饱腹感 |

---

### 9.3 心理卡

心理卡用于表达游戏的人文气质和可持续健康理念。

示例：

| 心理卡 | 效果 |
|---|---|
| 允许不完美 | 降低因不健康食物造成的饮食焦虑 |
| 不用一顿饭定义自己 | 抵消一次饮食失控后的心理惩罚，但不抵消营养负担 |
| 稳定比极端重要 | 若当天三餐都没有极端失衡，额外提升稳定度 |
| 今天已经很努力了 | 降低压力，恢复自控力 |
| 好好吃一顿正餐 | 若本餐包含主食、蛋白和蔬果，额外提升心情与稳定度 |

心理卡的作用是避免游戏变成“越克制越好”的数值系统。

---

## 10. 组合与牌型系统

游戏需要组合爽感，否则容易变成简单的加减分。

建议加入餐盘组合或生活组合。

---

### 10.1 均衡餐盘

条件：

- 主食 ≥ 1
- 蛋白 ≥ 1
- 蔬果 ≥ 1

效果：

- 饱腹感上升
- 饮食状态上升
- 稳定度上升
- 心情小幅上升

---

### 10.2 高纤维组合

条件：

- 全谷物 + 蔬菜 / 水果 / 豆类

效果：

- 饱腹感提升
- 下一餐冲动进食风险下降
- 长期饮食状态改善

---

### 10.3 高蛋白组合

条件：

- 两张以上优质蛋白食物

效果：

- 饱腹感提升
- 训练或高消耗角色恢复增强
- 如果缺少蔬果，饮食多样性可能下降

---

### 10.4 安慰餐

条件：

- 一张角色喜欢的高满足感食物
- 搭配至少一张稳定饮食状态的食物

示例：

- 奶茶 + 正餐
- 炸鸡 + 沙拉 + 无糖茶
- 火锅 + 豆腐 + 蔬菜

效果：

- 心情明显上升
- 饮食焦虑下降
- 饮食负担仍存在，但负面心理影响降低

设计意图：

> 喜欢的食物不是敌人，关键是如何放进整体生活里。

---

### 10.5 省钱健康餐

条件：

- 用低价格食材组成相对完整的一餐

示例：

- 米饭 + 鸡蛋 + 青菜
- 土豆 + 豆腐 + 番茄
- 燕麦 + 牛奶 + 香蕉

效果：

- 饮食状态上升
- 预算压力下降
- 在预算紧张场景中额外加成

---

### 10.6 外卖补救组合

条件：

- 使用外卖、速食或高负担食物后，搭配补救行动或食物

补救方式：

- 水果
- 蔬菜
- 喝水
- 早睡
- 第二餐清淡
- 散步或轻运动

效果：

- 降低后续失控风险
- 减少心理负担
- 部分降低饮食负担，但不应完全抵消

---

## 11. 生活事件系统

传统 Boss 不适合本游戏。挑战应做成隐性的生活环境变化，而不是人格化敌人。

这些事件不一定叫 Boss，可以叫：

- 生活事件
- 本周状态
- 环境压力
- 特殊情境
- 阶段规则

---

### 11.1 外卖优惠

描述：

> 平台发放满减券，外卖变得又便宜又方便。

效果：

- 外卖价格下降
- 外卖卡出现率上升
- 自炊吸引力下降
- 短期心情更容易恢复
- 饮食负担更难控制

---

### 11.2 考试周

适合作为困难模式。

效果：

- 每天压力自然上升
- 复习进度要求提高
- 奶茶、咖啡、速食出现率提高
- 早睡的机会成本提高
- 心情低时更容易出现摆烂行为

---

### 11.3 加班周

效果：

- 晚间时间减少
- 加班提高项目进度，但疲劳累积
- 外卖和便利店选择增加
- 运动和做饭成本上升

---

### 11.4 预算紧张

效果：

- 每日可用预算下降
- 高质量食材更难购买
- 鸡蛋、豆腐、米饭、土豆、应季菜价值提高
- 外卖优惠诱惑更强

---

### 11.5 聚餐邀请

玩家选择：

| 选择 | 效果 |
|---|---|
| 参加聚餐 | 心情和社交满足上升，饮食负担上升 |
| 拒绝聚餐 | 饮食更可控，但心情或社交满足下降 |
| 参加但控制选择 | 需要消耗自控力，获得平衡收益 |

---

### 11.6 情绪低谷

效果：

- 心情恢复变慢
- 甜食、零食、外卖出现率提高
- 心理卡效果增强
- 极端控制饮食的惩罚加重

---

### 11.7 冰箱清空 / 宿舍没存粮

效果：

- 自炊或简单食物减少
- 便利店、外卖、速食选项增加
- 采购行动价值提高

---

## 12. 难度设计

难度不应只是数值变高，而应通过生活条件变化实现。

| 难度 | 特点 |
|---|---|
| 普通 | 事件温和，预算适中，行动选择较多 |
| 困难 | 加入考试周、加班周、预算紧张等压力规则 |
| 专家 | 多个生活事件叠加，角色初始状态更差，结局要求更高 |

示例专家组合：

```text
角色：熬夜学生
场景：考试周
额外事件：外卖优惠 + 预算紧张 + 睡眠不足
```

---

## 13. 结局系统

结局不应只看单一分数，而应综合稳定度、饮食状态、心理状态、角色目标进度和预算情况。

示例结局类型：

| 结果 | 条件示例 | 说明 |
|---|---|---|
| 稳稳撑过 | 稳定度高，目标进度达标，饮食和心理状态都较好 | 角色找到了可持续节奏 |
| 勉强撑过 | 目标完成，但稳定度较低 | 靠硬扛完成，但留下隐患 |
| 吃得健康但心态崩了 | 饮食状态好，但心理状态差、压力高 | 计划过于极端，难以持续 |
| 心情不错但身体负担高 | 心理状态较好，但饮食负担高 | 短期舒服，长期仍需调整 |
| 预算撑住但状态不好 | 钱够用，但精力和饮食状态差 | 过度省钱造成生活质量下降 |
| 没撑住 | 稳定度过低或关键目标失败 | 生活节奏崩盘，需要重来 |

结局文本应尽量温和，不要羞辱玩家。

---

## 14. 长期成长与重复游玩

完整游戏需要有可重复游玩的动力。

可以包括：

### 14.1 解锁角色

玩家完成某些角色后，解锁新生活处境。

潜在角色：

- 熬夜学生
- 加班上班族
- 减脂上班族
- 健身大学生
- 情绪性进食者
- 独居青年
- 高血压中年人
- 素食新手
- 独居老人

涉及疾病、特殊人群时，应避免医疗化承诺，并强调这是生活策略游戏，不是治疗建议。

---

### 14.2 解锁卡牌

可解锁：

- 新食物
- 新菜谱
- 新行为
- 新心理卡
- 新生活事件

---

### 14.3 食谱图鉴

玩家打出过的组合可以进入图鉴。

例如：

- 番茄鸡蛋饭
- 豆腐青菜汤
- 燕麦酸奶碗
- 炸鸡补救餐
- 省钱学生餐
- 高纤维早餐

图鉴可以附带少量营养说明，但不应打断游戏节奏。

---

### 14.4 角色故事

每个角色可以有短剧情片段。

例如学生角色：

- 第一天：打开复习计划，发现时间不够
- 第三天：外卖满减和复习压力同时出现
- 第五天：第一次意识到自己不能只靠奶茶硬撑
- 结局：根据玩家选择形成不同生活状态

剧情不需要很长，但可以增加情感连接。

---

## 15. 风险与设计注意点

### 15.1 避免变成营养计算器

如果界面过多显示热量、蛋白、钠、糖、脂肪，游戏会变得像健康记录软件。

解决方式：

- 主界面只显示生活化指标
- 营养细节放到卡牌详情和图鉴中
- 用“饮食负担”“饱腹感”“身体状态”折叠复杂营养信息

---

### 15.2 避免健康道德化

不应把食物简单分成“好食物”和“坏食物”。

炸鸡、奶茶、方便面可以有高满足感、省时、省钱等正面价值，同时带来后续代价。

健康选择也可能有负面：

- 太贵
- 不满足
- 需要时间
- 需要精力
- 可能增加饮食焦虑

---

### 15.3 避免最优解单一

如果玩家发现“永远选某种健康组合就赢”，游戏会很快失去深度。

解决方式：

- 不同角色有不同权重
- 生活事件改变选择环境
- 同一食物对不同角色效果不同
- 预算、时间和精力共同限制行动
- 心理状态影响选择收益

---

### 15.4 避免过度复杂

虽然底层可以复杂，但玩家实时看到的系统应保持简洁。

建议主界面指标控制在 5–6 个，加一个角色专属指标即可。

---

## 16. 需要进一步评估的问题

希望评估者重点关注以下问题：

1. **核心玩法是否足够清晰？**  
   玩家扮演角色本人，在有限时间、预算和精力下撑过生活压力，这个定位是否成立？

2. **游戏是否容易从“生活策略”滑向“营养计算器”？**  
   当前指标和卡牌设计是否足够游戏化？

3. **稳定度作为总体指标是否合适？**  
   它是否能承担“生命值 / 状态值”的功能，又不会显得太医学化？

4. **经济系统是否必要？复杂度是否合适？**  
   预算、价格、采购和库存应该做到什么程度？

5. **不同角色的差异是否足够？**  
   是否应该进一步强化“角色能做什么 / 不能做什么”？

6. **生活事件作为隐性 Boss 是否有吸引力？**  
   外卖优惠、考试周、预算紧张、聚餐邀请等是否比传统 Boss 更适合本游戏？

7. **心理健康机制如何避免敏感或误导？**  
   如何表现压力、情绪进食、饮食焦虑，同时避免医学诊断或不当建议？

8. **卡牌组合是否有足够爽感？**  
   均衡餐盘、高纤维组合、安慰餐、外卖补救等牌型是否能带来类似构筑游戏的乐趣？

9. **单局时长应以 7 天还是 14 天为主？**  
   哪种更适合反复游玩和完整体验？

10. **游戏主题是否有市场辨识度？**  
    “撑过生活压力的饮食策略游戏”是否比“营养配餐卡牌游戏”更有吸引力？

---

## 17. 当前一句话总结

> **这是一款玩家扮演不同生活处境中的角色，在有限天数、预算、时间和精力下，通过饮食、行动和自我调节撑过一段压力时期的生活策略卡牌游戏。游戏用“稳定度”呈现总体状态，而稳定度由饮食、心理、疲劳、饱腹、睡眠、预算和角色目标等底层因素共同决定。**

---
